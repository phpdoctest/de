mhash_get_block_size

Gibt die Blockgröße von dem übergebenem Hash zurück

Beschreibung

int mhash_get_block_size ( int $hash )

mhash_get_block_size gibt die Größe des übergebenen Hashs.

mhash_get_block_size nimmt ein Argument, den Hash und gibt die Größe in Byte oder FALSE, wenn Hash nicht existiert.
