Einführung

Die SNMP Erweiterung ermöglicht es Remote-Devices über das Simple Network Management Protocol einfach zu verwalten.

Da diese Erweiterung um die Net-SNMP (oder UCD-SNMP auf älteren Systemen) Bibliothek herum implementiert ist, werden alle grundlegenden Konzepte von dieser übernommen. Auch ändern die PHP Funktionen ihr Verhalten abhängig von Net-SNMP Konfigurationsdateien und Umgebungsvariablen.

Seit PHP 5.4 ist eine objektorientierte API verfügbar.

Mehr Informationen über Net-SNMP können unter » http://www.net-snmp.org/ gefunden werden.
