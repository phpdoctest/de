bbcode_destroy

Schliesst eine vorhandene BBCode_container Ressource

Beschreibung

bool bbcode_destroy ( resource $bbcode_container )

Diese Funktion schliesst eine von bbcode_create geöffnete Ressource.

Parameter-Liste

bbcode_container

Eine BBCode_Container Ressource, die von bbcode_create zurückgegeben wird.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.
