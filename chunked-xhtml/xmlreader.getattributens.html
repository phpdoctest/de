XMLReader::getAttributeNs

Gibt den Wert eines Attributes nach lokalem Namen und URI zurück

Beschreibung

public string XMLReader::getAttributeNs ( string $localName , string $namespaceURI )

Der Wert des Attributes mit entsprechendem Namen und Namensraum (URI) oder eine leere Zeichenkette, wenn das Attribut nicht existiert oder der Cursor sich derzeitig nicht über einem Element befindet.

Parameter-Liste

localName

Der lokale Name des Attributes.

namespaceURI

Die URI des Namensraum.

Rückgabewerte

Der Wert des Attributes oder eine leere Zeichenkette (vor PHP 5.6) oder NULL (von PHP 5.6 an), wenn kein Attribut mit den Parametern localName und namespaceURI im aktuellen Element gefunden wurde oder der Cursor sich derzeitig nicht über einem Element befindet.

Changelog

Version Beschreibung
5.6.0 XMLReader::getAttributeNS gibt nun NULL zurück, wenn das Attribut nicht existiert.
