dio_truncate

Schneidet eine Datei ab

Beschreibung

bool dio_truncate ( resource $fd , int $offset )

dio_truncate schneidet eine Datei nach maximal offset ab.

War die Datei ursprünglich größer, so gehen die Daten verloren. War sie kleiner, ist nicht definiert, ob die Datei nicht geändert wird oder ob sie erweitert wird. Im letzteren Fall wird der erweiterte Teil mit Nullen aufgefüllt.

Parameter-Liste

fd

Ein von dio_open zurückgegebener Dateideskriptor.

offset

Der Offset in Bytes.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Anmerkungen

Hinweis: Diese Funktion ist auf Windows-Plattformen nicht implementiert.
