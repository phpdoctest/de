fdf_add_doc_javascript

Fügt JavaScript Code in ein FDF Dokument ein.

Beschreibung

bool fdf_add_doc_javascript ( resource $fdf_document , string $script_name , string $script_code )

Fügt ein Script zum FDF Dokument, welches Acrobat zu den doc-level Scripts eines Dokuments hinzufügt, sobald das FDF importiert wurde.

Parameter-Liste

fdf_document

Das FDF-Dokument-Handle, das von fdf_create, fdf_open oder fdf_open_string zurückgegeben wurde.

script_name

Der Scriptname.

script_code

Der Scriptcode. Es wird unbedingt empfohlen \r für Zeilenumbrüche innerhalb des Scriptcodes zu verwenden.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Beispiele

Beispiel #1 JavaScript-Code zu einem FDF hinzufügen

<?php
$fdf = fdf_create();
fdf_add_doc_javascript($fdf, "PlusOne", "function PlusOne(x)\r{\r  return x+1;\r}\r");
fdf_save($fdf);
?>
