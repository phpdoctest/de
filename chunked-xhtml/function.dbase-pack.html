dbase_pack

Entfernt die als gelöscht markierten Datensätze aus einer Datenbank

Beschreibung

bool dbase_pack ( int $dbase_identifier )

Entfernt die mittels dbase_delete_record als gelöscht markierten Datensätze endgültig aus der angegebenen Datenbank.

Parameter-Liste

dbase_identifier

Die Datenbank-Verbindungskennung, die von dbase_open oder dbase_create zurückgegeben wurde.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Beispiele

Beispiel #1 Leeren einer dBase-Datenbank

<?php

// im lesen-schreiben-Modus öffnen
$db = dbase_open('/tmp/test.dbf', 2);

if ($db) {
  $anzahl_datensaetze = dbase_numrecords($db);
  for ($i = 1; $i <= $anzahl_datensaetze; $i++) {
      dbase_delete_record($db, $i);
  }
  // Datensätze löschen
  dbase_pack($db);
}

?>
