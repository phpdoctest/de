Vordefinierte Konstanten

Folgende Konstanten werden von dieser Erweiterung definiert und stehen nur zur Verfügung, wenn die Erweiterung entweder statisch in PHP kompiliert oder dynamisch zur Laufzeit geladen wurde.

XSL_CLONE_AUTO (integer)
XSL_CLONE_NEVER (integer)
XSL_CLONE_ALWAYS (integer)
LIBXSLT_VERSION (integer)
Version der libxslt als Zahl, beispielsweise 10117. Verfügbar seit PHP 5.1.2.
LIBXSLT_DOTTED_VERSION (string)
Version der libxslt als String, beispielsweise 1.1.17. Verfügbar seit PHP 5.1.2.
LIBEXSLT_VERSION (integer)
Version der libexslt als Zahl, beispielsweise 813. Verfügbar seit PHP 5.1.2.
LIBEXSLT_DOTTED_VERSION (string)
Version der libexslt als String, beispielsweise 1.1.17. Verfügbar seit PHP 5.1.2.
XSL_SECPREF_NONE (integer)
Schaltet sämtliche Sicherheitsrestriktionen aus
XSL_SECPREF_READ_FILE (integer)
Verbietet den lesenden Zugriff auf Dateien
XSL_SECPREF_WRITE_FILE (integer)
Verbietet den schreibenden Zugriff auf Dateien
XSL_SECPREF_CREATE_DIRECTORY (integer)
Verbietet das Erzeugen von Verzeichnissen
XSL_SECPREF_READ_NETWORK (integer)
Verbietet den lesenden Zugriff auf Dateien über das Netzwerk
XSL_SECPREF_WRITE_NETWORK (integer)
Verbietet den schreibenden Zugriff auf Dateien über das Netzwerk
XSL_SECPREF_DEFAULT (integer)
Verbietet sämtliche Schreibzugriffe, identisch mit der Bitmaske von XSL_SECPREF_WRITE_NETWORK | XSL_SECPREF_CREATE_DIRECTORY | XSL_SECPREF_WRITE_FILE.
