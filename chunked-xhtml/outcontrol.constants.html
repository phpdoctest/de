Vordefinierte Konstanten

Die aufgelisteten Konstanten stehen immer zur Verfügung, da sie zum Grundbestand von PHP gehören.

PHP_OUTPUT_HANDLER_START (integer)

Zeigt an, dass die Pufferung der Ausgabe begann.

PHP_OUTPUT_HANDLER_WRITE (integer)

Zeigt an, dass der Ausgabepuffer geleert wird und Daten zur Ausgabe bereit stehen.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_FLUSH (integer)

Zeigt an, dass der Ausgabepuffer geleert und ausgegeben wurde.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_CLEAN (integer)

Zeigt an, dass der Ausgabepuffer gelöscht wurde.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_FINAL (integer)

Zeigt an, dass dies die letzte Operation des Ausgabepuffers ist.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_CONT (integer)

Zeigt an, dass der Ausgabepuffer geleert und ausgegeben wurde, aber die Pufferung fortgesetzt wird.

Zum Stand von PHP 5.4 ist dies ein Alias für PHP_OUTPUT_HANDLER_WRITE.

PHP_OUTPUT_HANDLER_END (integer)

Zeigt an, dass die Ausgabepufferung endete.

Zum Stand von PHP 5.4 ist dies ein Alias für PHP_OUTPUT_HANDLER_FINAL.

PHP_OUTPUT_HANDLER_CLEANABLE (integer)

Steuert, ob ein durch ob_start erzeugter Ausgabepuffer gelöscht werden kann.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_FLUSHABLE (integer)

Steuert, ob ein durch ob_start erzeugter Ausgabepuffer geleert und ausgegeben werden kann.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_REMOVABLE (integer)

Steuert, ob ein durch ob_start erzeugter Ausgabepuffer vor Ende der Skriptlaufzeit entfernt werden kann.

Verfügbar seit PHP 5.4.

PHP_OUTPUT_HANDLER_STDFLAGS (integer)

Die Standardeinstellungen der Pufferflags. Derzeit gleichbedeutend mit PHP_OUTPUT_HANDLER_CLEANABLE | PHP_OUTPUT_HANDLER_FLUSHABLE | PHP_OUTPUT_HANDLER_REMOVABLE.

Verfügbar seit PHP 5.4.
