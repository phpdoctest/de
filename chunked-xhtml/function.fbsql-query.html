fbsql_query

Sendet eine FrontBase-Abfrage

Beschreibung

resource fbsql_query ( string $query [, resource $link_identifier [, int $batch_size ]] )

Sendet eine Abfrage an die momentan aktive Datenbank auf dem Server.

Ist die Abfrage erfolgreich, können Sie fbsql_num_rows aufrufen, um herauszufinden wie viele Datensätze von einer SELECT-Abfrage zurückgeliefert wurden. Falls Sie eine DELETE-, INSERT-, REPLACE- oder UPDATE-Abfrage verwendet haben, können Sie fbsql_affected_rows verwenden, um die betroffenen Datensätze herauszufinden.

Parameter-Liste

query

Die auszuführende SQL-Abfrage.

Hinweis:

Der Abfrage-String soll immer mit einem Semikolon enden.

link_identifier

Eine von fbsql_connect oder fbsql_pconnect zurückgegebe FrontBase-Verbindung.

Wenn optional und nicht angegeben, wird versucht, eine bereits geöffnete FrontBase-Verbindung zu finden, falls keine vorhanden ist, wird versucht, eine zu öffnen, als ob fbsql_connect ohne Parameter aufgerufen wurde.

batch_size

Rückgabewerte

fbsql_query liefert zur Überprüfung des Erfolgs der Abfrage TRUE (nicht null) oder FALSE zurück. Ein TRUE-Rückgabewert bestätigt, dass die Abfrage gültig war und vom Server ausgeführt werden konnte. Über den Rückgabewert können nicht die betroffenen oder selektierten Datensätze ermittelt werden. Es ist sogar möglich, dass die Abfrage erfolgreich war, aber keine Datensätze betroffen oder selektiert werden.

Bei SELECT-Abfragen gibt fbsql_query eine neue Ergebniskennung zurück, die Sie an fbsql_result übergeben können.

fbsql_query schlägt ebenso fehl und liefert FALSE, wenn Sie keine Berechtigung zum Zugriff auf die verwiesene(n) Tabelle(n) haben.

Beispiele

Die folgende Abfrage ist syntaktisch falsch, so dass fbsql_query fehlschlägt und FALSE zurückliefert:

Beispiel #1 fbsql_query Beispiel

<?php
$result = fbsql_query("SELECT * WHERE 1=1")
    or die ("Ungültige Abfrage");
?>

Die folgende Abfrage ist sematisch falsch, wenn my_col keine Spalte in der Tabelle my_tbl ist. Deshalb schlägt fbsql_query fehl und liefert FALSE zurück:

Beispiel #2 fbsql_query Beispiel

<?php
$result = fbsql_query("SELECT my_col FROM my_tbl;")
    or die ("Ungültige Abfrage");
?>
