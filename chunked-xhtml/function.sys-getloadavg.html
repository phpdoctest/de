sys_getloadavg

Ermittelt durchschnittliche Systemlast

Beschreibung

array sys_getloadavg ( void )

Gibt drei Messwerte zurück die die durchschnittliche Systemlast (die Anzahl der Prozesse in der Ausführungswarteschlange) über die jeweils letzten 1, 5 und 15 Minuten representieren.

Rückgabewerte

Gibt ein array mit den drei Messwerten der letzten 1, 5 und 15 Minuten zurück.

Beispiele

Beispiel #1 sys_getloadavg Beispiel

<?php
$load = sys_getloadavg();
if ($load[0] > 80) {
    header('HTTP/1.1 503 Too busy, try again later');
    die('Serverlast zu hoch, versuchen Sie es später noch einmal.');
}
?>

Anmerkungen

Hinweis: Diese Funktion ist auf Windows-Plattformen nicht implementiert.
