fdf_next_field_name

Gibt den Namen des nächsten Feldes zurück

Beschreibung

string fdf_next_field_name ( resource $fdf_document [, string $fieldname ] )

Gibt den Namen des Feldes nach dem angegebenen Feld zurück. Dieser Name kann mit mehreren Funktionen verwendet werden.

Parameter-Liste

fdf_document

Das FDF-Dokument-Handle, das von fdf_create, fdf_open oder fdf_open_string zurückgegeben wurde.

fieldname

Name des FDF-Felds als Zeichenkette. Wenn nicht angegeben, wird das erste Feld angenommen.

Rückgabewerte

Gibt den Feldnamen als Zeichenkette zurück.

Beispiele

Beispiel #1 Alle Feldnamen in einem FDF ermitteln

<?php
$fdf = fdf_open($HTTP_FDF_DATA);
for ($field = fdf_next_field_name($fdf);
    $field != "";
    $field = fdf_next_field_name($fdf, $field)) {
    echo "field: $field\n";
}
?>
