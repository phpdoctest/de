dbase_delete_record

Markiert einen Datensatz als gelöscht

Beschreibung

bool dbase_delete_record ( int $dbase_identifier , int $record_number )

Markiert den angegebenen Datensatz in der Datenbank als gelöscht.

Hinweis:

Um den Datensatz wirklich aus der Datenbank zu entfernen, müssen sie noch dbase_pack aufrufen.

Parameter-Liste

dbase_identifier

Die Datenbank-Verbindungskennung, die von dbase_open oder dbase_create zurückgegeben wurde.

record_number

Eine Ganzzahl, die von 1 bis zur Anzahl der Datensätze in der Datenbank (wie von dbase_numrecords zurückgegeben) geht.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.
