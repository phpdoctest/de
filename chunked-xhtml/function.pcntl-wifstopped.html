pcntl_wifstopped

Gibt TRUE zurück, wenn der Kindprozess gerade gestoppt ist

Beschreibung

bool pcntl_wifstopped ( int $status )

Gibt TRUE zurück, wenn der Kindprozess der die Rückkehr verursachte momentan gestoppt ist; dies ist nur möglich, wenn der Aufruf von pcntl_waitpid mit der Option WUNTRACED geschah.

Der status-Parameter entspricht dem Statusparameter, der für einen erfolgreichen Aufruf von pcntl_waitpid verwendet wird.

Siehe auch pcntl_waitpid.
