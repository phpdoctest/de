Laufzeit-Konfiguration

Das Verhalten dieser Funktionen wird durch Einstellungen in der php.ini beeinflusst.

Readline Konfigurationsoptionen
Name Standard Veränderbar Changelog
cli.pager "" PHP_INI_ALL Verfügbar seit PHP 5.4.0.
cli.prompt "\\b \\> " PHP_INI_ALL Verfügbar seit PHP 5.4.0.

Hier eine kurze Erklärung der Konfigurationsoptionen:

cli.pager string

Externes Werkzeug für die Ausgabe der Kommandozeile.

cli.prompt string

Kommandozeile Eingabe.
