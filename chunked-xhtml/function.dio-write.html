dio_write

Schreibt Daten in eine Datei

Beschreibung

int dio_write ( resource $fd , string $data [, int $len = 0 ] )

Die Funktion dio_write schreibt bis zu len Bytes data auf den Dateideskriptor fd.

Parameter-Liste

fd

Ein von dio_open zurückgegebener Dateideskriptor.

data

Die zu schreibenden Daten.

len

Die Anzahl der zu schreibenden Bytes. Wenn nicht angegeben, werden die vollständigen Daten in die angegebene Datei geschrieben.

Rückgabewerte

Gibt die Anzahl der tatsächlich geschriebenen Bytes zurück.
