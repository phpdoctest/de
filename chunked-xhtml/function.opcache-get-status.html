opcache_get_status

Liefert den Status des OPcode Caches

Beschreibung

array opcache_get_status ([ boolean $get_scripts = TRUE ] )

Diese Funktion liefert Statusinformationen zu der Cache Instanz.

Parameter-Liste

get_scripts

Include script specific state information

Rückgabewerte

Liefert ein Array mit Informationen, optional enthält dieses Script spezifische Status-Informationen.

Fehler/Exceptions

Wenn opcache.restrict_api benutzt wird und der aktuelle Pfad eine Verletzung dieser Regel ist, wird eine E_WARNING Meldung geschmissen und keine Status-Informationen zurückgegeben.
