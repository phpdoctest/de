recode_file

Umkodierung von Dateien entsprechend der Recode-Anweisung

Beschreibung

bool recode_file ( string $request , resource $input , resource $output )

Umkodierung der Datei, die durch den Dateizeiger input referenziert wird, in die Datei output, entsprechend der Recode-Anweisung request.

Parameter-Liste

request

Der gewünschte Recode-Requesttyp.

input

Ein lokaler Dateizeiger vom Typ resource für den input-Parameter.

output

Ein lokaler Dateizeiger vom Typ resource für den output-Parameter.

Rückgabewerte

Gibt FALSE zurück, wenn es nicht möglich ist, die Anweisung durchzuführen, andernfalls TRUE.

Beispiele

Beispiel #1 Einfaches recode_file-Beispiel

<?php
$input = fopen('input.txt', 'r');
$output = fopen('output.txt', 'w');
recode_file("us..flat", $input, $output);
?>

Anmerkungen

Die Funktion verarbeitet derzeit keine Dateizeiger, die auf entfernte Dateien zeigen (URLs). Beide Dateizeiger müssen auf lokale Dateien zeigen.
