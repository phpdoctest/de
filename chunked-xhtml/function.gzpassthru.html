gzpassthru

Gibt alle verbleibenden Daten eines gz-Datei Deskriptors aus

Beschreibung

int gzpassthru ( resource $zp )

Liest bis EOF von der aktuellen Position auf dem übergebenen gz-Datei Deskriptor und schreibt die (unkomprimierten) Ergebnisse auf die Standard Ausgabe.

Hinweis:

Eventuell müssen sie gzrewind aufrufen, um den Datei Deskriptor auf den Anfang der Datei zurückzusetzen, falls sie schon Daten dorthin geschrieben haben.

Tipp

Wenn sie einfach den Inhalt einer Datei in den Ausgabe Puffer ausgeben wollen, ohne sie zuerst zu verändern oder zu einem bestimmten Offset zu springen (seek), können sie readgzfile nutzen, was den gzopen Aufruf spart.

Parameter-Liste

zp

Der gz-Datei Deskriptor. Er muss gültig sein und auf eine mit gzopen erfolgreiche geöffnete Datei zeigen.

Rückgabewerte

Die Anzahl umkomprimierter Zeichen gelesen von gz und durchgereicht zu der Eingabe oder FALSE bei Fehlern.

Beispiele

Beispiel #1 gzpassthru example

<?php
$fp = gzopen('file.gz', 'r');
gzpassthru($fp);
gzclose($fp);
?>
