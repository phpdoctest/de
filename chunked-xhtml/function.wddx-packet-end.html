wddx_packet_end

Schließt das WDDX-Paket mit der angegebenen ID

Beschreibung

string wddx_packet_end ( int $packet_id )

wddx_packet_end schließt das WDDX-Paket, welches mit packet_id identifiziert wird und gibt einen String, der das erzeugte WDDX-Paket enthält, zurück.
