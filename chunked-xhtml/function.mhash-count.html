mhash_count

Gibt die höchstmöglichen Hash-ID zurück

Beschreibung

int mhash_count ( void )

mhash_count gibt die höchste verfügbare Hash-ID zurück.

Rückgabewerte

Gibt die höchste verfügbare Hash-ID zurück. Hashes sind von 0 bis zu der Hash-ID nummeriert.

Beispiele

Beispiel #1 Alle Hashes durchlaufen

<?php

$nr = mhash_count();

for ($i = 0; $i <= $nr; $i++) {
    echo sprintf ("Die Blockgröße von %s ist %d\n",
        mhash_get_hash_name ($i),
        mhash_get_block_size ($i));
}
?>
