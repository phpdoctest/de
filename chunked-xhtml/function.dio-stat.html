dio_stat

Liefert Statusinformationen über einen Dateideskriptor

Beschreibung

array dio_stat ( resource $fd )

Die Funktion dio_stat gibt Informationen zum gegebenen Dateideskriptor zurück.

Parameter-Liste

fd

Ein von dio_open zurückgegebener Dateideskriptor.

Rückgabewerte

Gibt ein Array mit den folgenden Elementen zurück:

  • "device" - Device

  • "inode" - Inode

  • "mode" - Modus

  • "nlink" - Anzahl Links

  • "uid" - Benutzer Id

  • "gid" - Gruppentyp

  • "device_type" - Device-Typ

  • "size" - Größe in Bytes

  • "blocksize" - Blockgröße

  • "blocks" - Anzahl allozierter Blöcke

  • "atime" - Letzter Zugriff

  • "mtime" - Letze Modifikation

  • "ctime" - Letzte Änderung

Im Fehlerfall liefert dio_stat NULL zurück.
