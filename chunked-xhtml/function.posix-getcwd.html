posix_getcwd

Pfadname des aktuellen Verzeichnisses

Beschreibung

string posix_getcwd ( void )

Gibt den absoluten Pfadnamen des aktuellen Arbeitsverzeichnisses des Skripts zurück. Falls ein Fehler auftritt, wird errno (Fehlernummer) gesetzt, das mittels posix_get_last_error abgerufen werden kann.

Rückgabewerte

Gibt bei Erfolg den absoluten Pfadnamen als Zeichenkette zurück. Falls ein Fehler auftritt, wird FALSE zurückgegeben und errno gesetzt, das mittels posix_get_last_error überprüft werden kann.

Beispiele

Beispiel #1 posix_getcwd-Beispiel

Dieses Beispiel gibt den absoluten Pfad des aktuellen Arbeitsverzeichnisses des Skripts aus.

<?php
echo 'Das aktuelle Arbeitsverzeichnis ist '.posix_getcwd();
?>

Anmerkungen

Hinweis:

Diese Funktion schlägt fehl, wenn

  • es keine Lese- oder Suchberechtigung gibt
  • der Pfadname nicht mehr existiert
