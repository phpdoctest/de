filepro_rowcount

bestimmt, wieviele Zeilen eine filePro-Datenbank enthält

Beschreibung

int filepro_rowcount ( void )

Gibt die Anzahl der Zeilen in der geöffneten filePro-Datenbank zurück.

Hinweis: Wenn Safe Mode aktiviert ist, überprüft PHP, ob die Dateien/Verzeichnisse, die mit dem Skript bearbeitet werden sollen, die gleiche UID (Eigentümer) haben wie das Skript selbst.

Rückgabewerte

Gibt die Anzahl der Zeilen in der geöffneten filePro-Datenbank oder im Fehlerfall FALSE zurück.
