pspell_config_mode

Ändert den Modus für die Anzahl gelieferter Vorschläge

Beschreibung

bool pspell_config_mode ( int $dictionary_link , int $mode )

pspell_config_mode sollte auf die Konfiguration angewendet werden, bevor pspell_new_config aufgerufen wird. Diese Funktion bestimmt, wie viele Vorschläge von pspell_suggest geliefert werden.

Parameter-Liste

dictionary_link

mode

Der Modus, in dem die Rechtschreibprüfung arbeitet. Es stehen mehrere Modi zur Verfügung:

  • PSPELL_FAST - Schneller Modus (geringste Anzahl Vorschläge)
  • PSPELL_NORMAL - Normaler Modus (mehr Vorschläge)
  • PSPELL_BAD_SPELLERS - Langsamer Modus (viele Vorschläge)

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Beispiele

Beispiel #1 pspell_config_mode-Beispiel

<?php
$pspell_config = pspell_config_create("de");
pspell_config_mode($pspell_config, PSPELL_FAST);
$pspell_link = pspell_new_config($pspell_config);
pspell_check($pspell_link, "dieSpinne");
?>
