HP-UX spezifische Installationsnotizen

Dieser Abschnitt entält einige Notizen und Hinweise zur Installation von PHP auf HP-UX Systemen.

Es gibt hauptsächlich zwei Möglichkeiten PHP auf einem HP-UX System zu installieren. Entweder kompilieren sie PHP selber oder Sie benutzen bereits vorkompilierten Binärpakete.

Offizielle vorkompilierte Pakete können hier bezogen werden: » http://software.hp.com/

Bis dieser Abschnitt nicht neu geschrieben wurde, ist die Anleitung für das Kompilieren von PHP (und seinen Erweiterungen) auf HP-UX Systemen bis auf Weiteres entfernt. Sie können Sich jedoch auf extern Resourcen über den Prozess des Kompilieren informieren: » Building Apache and PHP on HP-UX 11.11
