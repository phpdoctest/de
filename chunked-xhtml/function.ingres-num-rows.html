ingres_num_rows

Gibt die Anzahl der Datensätze der Ergebnismenge oder die Anzahl der von der letzten Abfrage betroffenen Datensätze zurück

Beschreibung

int ingres_num_rows ( resource $result )

Diese Funktion dient hauptsächlich dazu, die Anzahl der veränderten Datensätze in einer Datenbank festzustellen. Sie kann aber auch dazu verwendet werden, die Anzahl der Datensätze zu ermitteln, die von einem SELECT geholt werden.

Hinweis:

Falls scrollable cursors deaktiviert ist und diese Funktion aufgerufen wird, bevor ingres_fetch_array, ingres_fetch_object, oder ingres_fetch_row aufgerufen werden, löscht der Server die Ergebnisdaten so dass das Skript sie nicht mehr auslesen kann.

Sie sollten stattdessen die Daten der Ergebnismenge mit einer dieser fetch-Funktionen in einer Schleife auslesen, solange, bis keine Datensätze mehr zum Lesen da sind und damit die Schleifenbedingung FALSE wird.

Parameter-Liste

result

Die Ergebniskennung

Rückgabewerte

Bei delete-, insert- or update-Abfragen, gibt ingres_num_rows die Anzahl der durch diese Abfragen betroffenen Datensätze zurück. Bei anderen Abfragen gibt ingres_num_rows die Anzahl der Datensätze in der Ergebnismenge zurück.
