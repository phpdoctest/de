XSLTProcessor::transformToURI

Transformierung zu einer URI

Beschreibung

int XSLTProcessor::transformToURI ( DOMDocument $doc , string $uri )

Transformiert die Quell-Node unter Verwendung des mittels XSLTProcessor::importStylesheet-Methode importierten Stylesheets zu einer gegebene URI.

Parameter-Liste

doc

Das zu verarbeitende Dokument.

uri

Die URI für das Ziel der Transformation.

Rückgabewerte

Liefert die Anzahl geschriebener Bytes zurück oder FALSE falls ein Fehler aufgetreten ist.

Beispiele

Beispiel #1 Transformierung in eine HTML-Datei

<?php

// XML-Quellen laden
$xml = new DOMDocument;
$xml->load('collection.xml');

$xsl = new DOMDocument;
$xsl->load('collection.xsl');

// Prozessor instanziieren und konfigurieren
$proc = new XSLTProcessor;
$proc->importStyleSheet($xsl); // XSL Document importieren

$proc->transformToURI($xml, 'file:///tmp/out.html');

?>
