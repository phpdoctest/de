mime_content_type

Ermittelt den MIME-Typ des Inhalts einer Datei

Beschreibung

string mime_content_type ( string $filename )

Gibt den MIME-Inhaltstyp für eine Datei unter verwendung von Informationen von der magic.mime-Datei.

Parameter-Liste

filename

Pfad zur zu testenden Datei

Rückgabewerte

Gibt den Inhaltstxp im MIME-Format zurück, wie z.B. text/plain oder application/octet-stream.

Beispiele

Beispiel #1 mime_content_type Example

<?php
echo mime_content_type('php.gif') . "\n";
echo mime_content_type('test.php');
?>

Das oben gezeigte Beispiel erzeugt folgende Ausgabe:

image/gif
text/plain
