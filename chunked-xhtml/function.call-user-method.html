call_user_method

Ruft eine benannte Methode eines Objekts auf

Warnung

This function was DEPRECATED in PHP 4.1.0, and REMOVED in PHP 7.0.0.

Alternatives to this function include:

  • call_user_func

Beschreibung

mixed call_user_method ( string $method_name , object &$obj [, mixed $parameter [, mixed $... ]] )
Warnung

Diese Funktion ist seit PHP 4.1.0 DEPRECATED (veraltet) und seit PHP 7.0.0 ENTFERNT.

Parameter-Liste

method_name

Der Name der aufgerufenen Methode.

obj

Das object dessen method_name aufgerufen wird.

parameter ...

Die optionalen Parameter.

Beispiele

Beispiel #1 call_user_function Alternative

<?php
call_user_func(array($obj, $method_name), $parameter /* , ...*/);
call_user_func(array(&$obj, $method_name), $parameter /* , ...*/); // PHP4
?>
