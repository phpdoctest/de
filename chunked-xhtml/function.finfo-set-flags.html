finfo_set_flags

finfo::set_flags

Setzt libmagic Konfigurationsoptionen

Beschreibung

Prozeduraler Stil

bool finfo_set_flags ( resource $finfo , int $options )

Objektorientierter Stil

public bool finfo::set_flags ( int $options )

Diese Funktion setzt verschiedene Fileinfo Optionen. Optionen können auch direkt in finfo_open oder anderen Fileinfo-Funktionen gesetzt werden.

Parameter-Liste

finfo

Die von der Funktion finfo_open zurückgegebene Fileinfo-Ressource.

options

Eine oder eine Verbindung von mehreren Fileinfo -Konstanten.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.
