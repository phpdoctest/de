token_name

Gibt Bezeichner für ein PHP-Token zurück

Beschreibung

string token_name ( int $token )

token_name gibt den Bezeichner für den Wert von token zurück.

Parameter-Liste

token

Der Tokenwert.

Rückgabewerte

Der Bezeichner für den angegebenen token.

Beispiele

Beispiel #1 token_name Beispiele

<?php
// 260 ist der Tokenwert für den T_EVAL-Token
echo token_name(260);        // -> "T_EVAL"

// Eine Tokenkonstante liefert den eigenen Namen zurück
echo token_name(T_FUNCTION); // -> "T_FUNCTION"
?>
