dio_seek

Setzt den Dateizeiger

Beschreibung

int dio_seek ( resource $fd , int $pos , int $whence )

Die Funktion dio_seek setzt den Dateizeiger der Datei fd.

Gibt an, wie die Position pos interpretiert werden soll:

  • SEEK_CUR - gibt an, dass pos die Position relativ von der momentanen Dateizeigerposition ist.

  • SEEK_END - gibt an, dass pos die Position relativ zum Dateiende ist. Eine negative Zahl setzt den Dateizeiger innerhalb der Datei, ein positiver Wert bewirkt, dass die Datei vom Ende bis zur neuen Position mit Nullen aufgef端llt wird.
