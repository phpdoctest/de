fdf_create

Erzeugt ein neues FDF Dokument

Beschreibung

resource fdf_create ( void )

Erzeugt ein neues FDF Dokument.

Diese Funktion wird benötigt, wenn man Eingabefelder in einem PDF Dokument mit Daten belegen möchte.

Rückgabewerte

Gibt ein FDF-Dokument-Handle zurück, oder FALSE im Fehlerfall.

Beispiele

Beispiel #1 Füllen von Formularwerten in einem PDF Dokument

          
<?php
$outfdf = fdf_create();
fdf_set_value($outfdf, "band", $band, 0);

fdf_set_file($outfdf, "http:/testfdf/resultlabel.pdf");
fdf_save($outfdf, "outtest.fdf");
fdf_close($outfdf);
Header("Content-type: application/vnd.fdf");
$fp = fopen("outtest.fdf", "r");
fpassthru($fp);
unlink("outtest.fdf");
?>
