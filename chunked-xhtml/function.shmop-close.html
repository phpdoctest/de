shmop_close

Schließt einen gemeinsamen Speicherblock

Beschreibung

void shmop_close ( int $shmid )

shmop_close wird benutzt, um einen gemeinsamen Speicherblock zu schließen.

Parameter-Liste

shmid

Der Bezeichner für den gemeinsamen Speicherblock, der von shmop_open erzeugt wurde.

Rückgabewerte

Es wird kein Wert zurückgegeben.

Beispiele

Beispiel #1 Einen gemeinsamen Speicherblock schließen

<?php
shmop_close($shm_id);
?>

In diesem Beispiel wird der gemeinsame Speicherblock $shm_id wieder geschlossen.
