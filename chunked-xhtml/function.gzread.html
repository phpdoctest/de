gzread

Liest binary-safe aus einer gz-Datei

Beschreibung

string gzread ( resource $zp , int $length )

gzread liest bis zu length Bytes aus der gegebenen gz-Datei Resource. Lesen stoppt, wenn length (unkomprimierte) Bytes gelesen wurden oder EOF (Dateiende) erreicht wurde, je nachdem, was zuerst eintrifft.

Parameter-Liste

zp

Die gz-Datei Ressource. Dies muss auf eine gültige, erfolgreich mit gzopen geöffnete Datei zeigen.

length

Die Anzahl der zu lesenden Bytes.

Rückgabewerte

Die gelesenen Daten.

Beispiele

Beispiel #1 gzread Beispiel

<?php
// Liest den Inhalt einer gz-Datei in einen String
$filename = "/usr/local/something.txt.gz";
$zd = gzopen($filename, "r");
$contents = gzread($zd, 10000);
gzclose($zd);
?>
