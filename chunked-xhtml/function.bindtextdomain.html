bindtextdomain

Setzt den Pfad für eine Domain

Beschreibung

string bindtextdomain ( string $domain , string $directory )

Die Funktion bindtextdomain setzt den Pfad für eine Domain. Sie gibt den vollständigen Pfad der Domain zurück, der gerade gesetzt wurde.

Parameter-Liste

domain

Die Domain

directory

Der Verzeichnispfad

Rückgabewerte

Der vollständige Pfadname für die aktuell gesetzte domain.

Beispiele

Beispiel #1 bindtextdomain-Beispiel

<?php

$domain = 'MeineApplikation';
echo bindtextdomain($domain, '/usr/share/MeineApplikation/locale');

?>

Das oben gezeigte Beispiel erzeugt folgende Ausgabe:

/usr/share/MeineApplikation/locale
