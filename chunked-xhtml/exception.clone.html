Exception::__clone

Klont die Exception

Beschreibung

final private void Exception::__clone ( void )

Versucht, die Exception zu klonen. Dies erzeugt einen Fatal Error.

Parameter-Liste

Diese Funktion hat keine Parameter.

Rückgabewerte

Es wird kein Wert zurückgegeben.

Fehler/Exceptions

Exceptions sind nicht klonbar.
