readgzfile

Gibt eine gz-komprimierte Datei aus

Beschreibung

int readgzfile ( string $filename [, int $use_include_path = 0 ] )

Liest eine Datei, dekomprimiert sie und gibt das Ergebnis auf die Standardausgabe aus.

readgzfile kann auch Dateien lesen die nicht im gzip Format vorliegen; in diesem Fall wird der Inhalt der Datei direkt ohne Dekomprimierung ausgegeben.

Parameter-Liste

filename

Der Dateiname. Die Datei wird geöffnet und ihr Inhalt auf die Standartausgabe ausgegeben.

use_include_path

Sie können diesen Parameter auf 1 setzen wenn die Datei auch im include_path gesucht werden soll.

Rückgabewerte

Gibt die Anzahl der ausgegebenen Zeichen zurück. Falls ein Fehler auftritt wird FALSE zurückgegeben, außerdem wird eine Fehlermeldung ausgegeben es sei denn die Funktion wurde als @readgzfile aufgerufen. printed.
