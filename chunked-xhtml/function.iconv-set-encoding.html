iconv_set_encoding

Einstellungen für die Zeichensatzkonvertierung setzen

Beschreibung

bool iconv_set_encoding ( string $type , string $charset )

Die Funktion ändert den Wert von type in charset.

Parameter-Liste

type

Der Wert von type kann einer der folgenden sein:

  • input_encoding
  • output_encoding
  • internal_encoding

charset

Der Zeichensatz.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Beispiele

Beispiel #1 iconv_set_encoding Beispiel:

<?php
iconv_set_encoding("internal_encoding", "UTF-8");
iconv_set_encoding("output_encoding", "ISO-8859-1");
?>
