ingres_pconnect

Öffnet eine persistente Verbindung zu einer Ingres-Datenbank

Beschreibung

resource ingres_pconnect ([ string $database [, string $username [, string $password [, array $options ]]]] )

Öffnet eine persistente Verbindung zu einer Ingres-Datenbank

Es gibt nur zwei Unterschiede zwischen dieser Funktion und ingres_connect: Zum ersten: beim Verbindungsaufbau schaut die Funktion zuerst, ob schon eine geöffnete (persistente) Verbindung besteht, die mit denselben Parametern geöffnet wurde. Falls eine gefunden wird, wird eine Verbindungskennung zurückgegeben, anstatt eine neue Verbindung zu öffnen. Zum zweiten: Die Verbindung zum Ingres-Server wird nicht geschlossen wenn die Ausführung des Skripts beendet ist, sondern die Verbindung bleibt zum weiteren Gebrauch offen (ingres_close schließt die Verbindungen, die mit ingres_pconnect geöffnet wurden nicht). Darum wird diese Art der Verbindung 'persistent' genannt.

Parameter-Liste

database

Der Datenbankname. Dieser muss die folgende Syntax haben:

[vnode::]dbname[/svr_class]

username

Der Ingres-Benutzername

password

Das Passwort von username

options

Bei ingres_connect finden Sie eine Liste der Optionen, die übergeben werden können.

Rückgabewerte

Bei Erfolg wird eine Ingres-Verbindungskennung zurückgegeben. Im Fehlerfall wird FALSE zurückgegeben.
