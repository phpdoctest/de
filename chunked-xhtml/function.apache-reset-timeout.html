apache_reset_timeout

Setzt den Apache Write-Time zurück

Beschreibung

bool apache_reset_timeout ( void )

apache_reset_timeout setzt den Apache Write-Timer zurück, der standardmäßig auf 300 Sekunden eingestellt ist. Mit set_time_limit(0); ignore_user_abort(true) und periodischen apache_reset_timeout-Aufrufen kann Apache theoretisch dauerhaft laufen.

Diese Funktion erfordert Apache 1.

Rückgabewerte

Gibt bei Erfolg TRUE zurück. Im Fehlerfall wird FALSE zurückgegeben.

Anmerkungen

Hinweis: Diese Funktion steht im Safe Mode nicht zur Verfügung.
