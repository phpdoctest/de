dcgettext

Überschreibt die gesetzte Domain für eine einzelne Abfrage

Beschreibung

string dcgettext ( string $domain , string $message , int $category )

Die Funktion erlaubt es Ihnen, die aktuell gesetzte Domain für eine einzelne Abfrage zu überschreiben.

Parameter-Liste

domain

Die Domain

message

Die Mitteilung

category

Die Kategorie

Rückgabewerte

Bei Erfolg ein string.
