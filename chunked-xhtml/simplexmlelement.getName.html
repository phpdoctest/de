SimpleXMLElement::getName

Ermittelt den Namen eines XML-Elements

Beschreibung

SimpleXMLElement
string getName ( void )

Ermittelt den Namen eines XML-Elements.

Rückgabewerte

Die getName-Methode gibt den Namen des durch das SimpleXMLElement-Objekt referenzierte XML-Tag als String zurück.

Beispiele

Beispiel #1 Ermitteln des XML-Element-Namens

<?php

$sxe = new SimpleXMLElement($xmlstr);

echo $sxe->getName() . "\n";

foreach ($sxe->children() as $child)
{
    echo $child->getName() . "\n";
}

?>
