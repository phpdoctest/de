Einführung

Die PDF Funktionen in PHP können PDF Dateien unter Verwendung der PDFlib Bibliothek von PDFlib GmbH (» www.pdflib.com) erzeugen. Eine eingeschränkte Version namens PDFlib Lite 7 ist kostenlos verfügbar, wird aber seit 2010 nicht weiter gepflegt. Die PDFlib Integration in PHP wird noch immer von PDFlib GmbH aktiv gepflegt. Sie ist hauptsächlich auf die kommerziellen PDFlib Versionen gerichtet, aber kann auch für die alte PDFlib Lite genutzt werden.

Zum schnellen Einstieg sollten Sie sich unbedingt die Code-Beispiele, die in jedem PDFlib-Paket enthalten sind, und die umfangreichen Handbücher im PDF-Format, die ebenfalls enthalten sind, ansehen. Zumindest sollten Sie einen Blick auf das hello.php Beispiel werfen, um die grundlegende Einrichtung für die PDFlib Verwendung zu erlernen.

Hinweis:

Wenn Sie an weiterer frei verfügbarer Software zur PDF-Erzeugung interessiert sind, die keine externe PDF-Bibliothek verwendet, können Sie sich in dieser FAQ informieren.
