dio_close

Schließt eine Datei

Beschreibung

void dio_close ( resource $fd )

Die Funktion dio_close schließt die Datei fd.

Parameter-Liste

fd

Eine von dio_open zurückgegebene Dateideskriptorresource.

Rückgabewerte

Es wird kein Wert zurückgegeben.

Beispiele

Beispiel #1 Schließen eines geöffneten Dateideskriptors

<?php
$fd = dio_open('/dev/ttyS0', O_RDWR);

dio_close($fd);
?>
