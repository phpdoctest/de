Laufzeit-Konfiguration

Das Verhalten dieser Funktionen wird durch Einstellungen in der php.ini beeinflusst.

Mcrypt Konfigurationsoptionen
Name Standard Veränderbar Changelog
mcrypt.algorithms_dir NULL PHP_INI_ALL  
mcrypt.modes_dir NULL PHP_INI_ALL  
Weitere Details und die Definitionen der PHP_INI_*-Konstanten finden Sie im Wo Konfigurationseinstellungen gesetzt werden können.

Hier eine kurze Erklärung der Konfigurationsoptionen:

mcrypt.algorithms_dir string

Das Verzeichnis, das den Algorythmus enthält. Entspricht standardmäßig den innerhalb von libmcrypt kompilierten Verzeichnissen, welches in der Regel /usr/local/lib/libmcrypt ist. Siehe mcrypt_list_algorithms für genauere Informationen.

mcrypt.modes_dir string

Das Verzeichnis, das die Modi enthält. Entspricht standardmäßig den innerhalb von libmcrypt kompilierten Verzeichnissen, welches in der Regel /usr/local/lib/libmcrypt ist. Siehe mcrypt_list_modes für zusätzliche Informationen.
