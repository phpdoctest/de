posix_getpgid

Liefert die Prozessgruppenkennung (Process Group ID) für die Job-Kontrolle

Beschreibung

int posix_getpgid ( int $pid )

Gibt die Prozessgruppenkennung des Prozzesses pid zurück.

Parameter-Liste

pid

Die Prozess-ID

Rückgabewerte

Gibt die Kennung als Ganzzahl zurück. Im Fehlerfall wird FALSE zurückgegeben.

Beispiele

Beispiel #1 posix_getpgid-Beispiel

<?php
$pid = posix_getppid();
echo posix_getpgid($pid); //35
?>

Anmerkungen

Hinweis:

Dies ist keine POSIX-Funktion, aber sie ist auf BSD- und System V-Systemen gängig. Falls Ihr System diese Funktion nicht unterstützt, wird sie während der Übersetzung nicht eingebunden. Ob es die Funktion gibt, kann mittels function_exists überprüft werden.
