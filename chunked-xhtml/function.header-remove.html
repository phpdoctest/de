header_remove

Remove previously set headers

Beschreibung

void header_remove ([ string $name ] )

Removes an HTTP header previously set using header.

Parameter-Liste

name

The header name to be removed.

Hinweis: This parameter is case-insensitive.

Rückgabewerte

Es wird kein Wert zurückgegeben.

Beispiele

Beispiel #1 Unsetting specific header.

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove("X-Foo"); 
?>

Das oben gezeigte Beispiel erzeugt eine ähnliche Ausgabe wie:

X-Bar: Baz

Beispiel #2 Unsetting all previously set headers.

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove(); 
?>

Das oben gezeigte Beispiel erzeugt eine ähnliche Ausgabe wie:


    

   

  

 



 

  
Anmerkungen

  
Achtung
   

    This function will remove all headers set by PHP,
    including cookies, session and the X-Powered-By headers.
   

  

  
Hinweis: 
 

  Header sind nur dann zugänglich und werden nur dann gesendet,
  wenn die genutzte SAPI sie unterstützt.
