Anforderungen

Um eine Datenbank über die dbx Extension ansprechen zu können muss die entsprechende Datenbankextension entweder in PHP einkompiliert oder geladen sein und von dbx unterstützt werden. Zur Zeit werden folgende Datenbanken unterstützt, andere werden folgen:

  • FrontBase (Verfügbar ab PHP 4.1.0).
  • Microsoft SQL Server
  • MySQL
  • ODBC
  • PostgreSQL
  • Sybase-CT (Verfügbar ab PHP 4.2.0).
  • Oracle (oci8) (Verfügbar ab PHP 4.3.0).
  • SQLite (PHP 5).

Die zum Hinzufügen weiterer Datenbanken nötigen Informationen können unter » http://www.guidance.nl/php/dbx/doc/ gefunden werden.
