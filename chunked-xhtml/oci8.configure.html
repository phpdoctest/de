Installation

PHP muss mit der option --with-oci8[=DIR] kompiliert werden, wobei DIR standardmässig auf die Umgebungsvariable ORACLE_HOME gesetzt ist.

Wenn der » Oracle Instant Client benutzt wird, muss PHP mit der option --with-oci8=instantclient,/path/to/oic/lib erstellt werden. Es ist zu beachten, dass der Oracle Instant Client erst seit der Version 4.3.11 und 5.0.4 unterstützt wird.

Hinweis:

Es ist darauf zu achten, dass PHP mit dem Oracle Instant Client bei den OCI-Versionen vor 1.1 mit der Option --with-oci8-instant-client konfiguriert werden muss.
