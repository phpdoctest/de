Einführung

Die password hashing API stellt einen einfach zu benutzenden Wrapper über der funktion crypt zur einfachen Erstellung und Verwaltung von Passwörtern zur Verfügung.

Diese Erweiterung ist verfügbar seit PHP 5.5.0 aber es gibt ebenso eine » userland implementation für PHP >= 5.3.7.
