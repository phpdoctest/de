Liste weiterer reservierter Wörter

Diese Wörter können nicht verwendet werden um eine Klasse, ein Interface oder einen Trait zu benennen. Diese Wörter können ebenfalls nicht in Namespaces verwendet werden. Einige der Wörter sind reserviert, um zukünftige PHP Features zu ermöglichen ohne die Abwärtskompatibilität zu brechen.

Reservierte Wörter
int (ab PHP 7) float (ab PHP 7) bool (ab PHP 7) string (ab PHP 7)
true (ab PHP 7) false (ab PHP 7) null (ab PHP 7) resource (ab PHP 7)
object (ab PHP 7) mixed (ab PHP 7) numeric (ab PHP 7)
