Laufzeit-Konfiguration

Das Verhalten dieser Funktionen wird durch Einstellungen in der php.ini beeinflusst.

htscanner Konfigurationsoptionen
Name Standard Veränderbar Changelog
htscanner.config_file ".htscanner" PHP_INI_SYSTEM  
htscanner.default_docroot "/" PHP_INI_SYSTEM  
htscanner.default_ttl "300" PHP_INI_SYSTEM  
htscanner.stop_on_error "Off" PHP_INI_SYSTEM  
Weitere Details und die Definitionen der PHP_INI_*-Konstanten finden Sie im Wo Konfigurationseinstellungen gesetzt werden können.

Hier eine kurze Erklärung der Konfigurationsoptionen:

htscanner.config_file string

Dateiname der zu nutzenden Konfigurationsdatei.

htscanner.default_docroot string

Standardmäßiger Datei-Root.

htscanner.default_ttl int

Cache-Zeit für die Konfigurationsdaten in Sekunden

htscanner.stop_on_error int

Bei einem Fehler anhalten(Parse-Fehler, kann keine INI-Einstellung setzen).
