ftp_get_option

Ruft diverse Laufzeitoptionen des angegebenen FTP-Streams ab

Beschreibung

mixed ftp_get_option ( resource $ftp_stream , int $option )

Diese Funktion gibt den Wert der angeforderten Option option von der angegebenen FTP-Verbindung zurück.

Parameter-Liste

ftp_stream

Der Verbindungshandler der FTP-Verbindung.

option

Zur Zeit werden die folgenden Optionen unterstützt:

Unterstützte FTP-Laufzeitoptionen
FTP_TIMEOUT_SEC Gibt den aktuellen Timeout für alle Netzwerkoperationen zurück.
FTP_AUTOSEEK Gibt TRUE zurück, wenn diese Option aktiv ist, sonst FALSE.

Rückgabewerte

Gibt den Wert bei Erfolg zurück oder FALSE, wenn die angegebene Option option nicht unterstützt wird. Im letzteren Fall wird zusätzlich eine Warnung geworfen.

Beispiele

Beispiel #1 ftp_get_option-Beispiel

<?php
// Bestimme den Timeout des angegebenen FTP-Streams
$timeout = ftp_get_option($conn_id, FTP_TIMEOUT_SEC);
?>
