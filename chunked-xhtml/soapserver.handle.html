SoapServer->handle()

Verarbeitet SOAP-Anfragen

Beschreibung

public void SoapServer::handle ([ string $soap_request ] )

Verarbeitet eine SOAP-Anfrage, ruft benötigte Funktionen auf und schickt eine Antwort zurück.

Parameter-Liste

soap_request

SOAP-Anfrage. Wenn das Argument nicht gegeben ist, werden die originalen (raw) POST Daten der HTTP Anfrage genutzt.

Rückgabewerte

Es wird kein Wert zurückgegeben.

Beispiele

Beispiel #1 Beispiele

<?php
function test($x)
{
    return $x;
}

$server = new SoapServer(null, array('uri' => "http://test-uri/"));
$server->addFunction("test");
$server->handle();
?>
