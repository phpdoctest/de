hash_hmac_file

Berechnet einen Hash einer Datei mit Schlüssel unter Verwendung von HMAC

Beschreibung

string hash_hmac_file ( string $algo , string $filename , string $key [, bool $raw_output = false ] )

Parameter-Liste

algo

Name des gewählten Hash-Algorithmus (z.B. "md5", "sha256", "haval160,4", usw...). Für eine Liste der unterstützten Algorithmen, siehe hash_algos.

filename

URL der Datei, die gehasht werden soll, fopen-Wrapper werden unterstützt.

key

Geheimer Schlüssel für die Berechnung der HMAC-Variante des Hashs.

raw_output

Ist dieser Parameter TRUE, werden direkt Binärdaten zurückgegeben, andernfalls werden klein geschriebene Hexadezimalziffern zurückgegeben.

Rückgabewerte

Gibt den berechneten Hash als Hexadezimalzahl zurück, außer raw_output ist wahr, in diesem Fall wird die binäre Darstellung des Hashes zurückgegeben.

Beispiele

Beispiel #1 hash_hmac_file-Beispiel

<?php
/* Erstelle die Datei, deren Inhalt gehasht werden soll */
file_put_contents('example.txt', 'Franz jagt im komplett verwahrlosten Taxi quer durch Bayern.');

echo hash_hmac_file('md5', 'example.txt', 'geheimnis');
?>

Das oben gezeigte Beispiel erzeugt folgende Ausgabe:

e949d0086c79cc6d67cf757bab4e5be8
